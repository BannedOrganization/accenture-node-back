### СРЕДА ЗАПУСКА BACKEND application
------------
1) NodeJS 14.17+
2) MongoDB latest version

### УСТАНОВКА
------------
1. git clone https://github.com/BannedOrganization/accenture-node-back
2. cd accenture-node-back
3. npm i
4. npm run build
5. npm run start


### Конфигурация
В файле .env укажите пути до:
1. Mongo DB: DATABASE_HOST
